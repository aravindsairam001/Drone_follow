Screenshots Directory

Place your drone system screenshots in this directory for inclusion in the README.md documentation.

Suggested screenshots to include:
1. drone_view.jpg - Main interface showing tracking and status information
2. human_detection.jpg - YOLOv8 detection with confidence scores
3. control_ui.jpg - UI elements showing drone status and controls
4. follow_mode.jpg - Drone actively following a target
5. tracking_interface.jpg - Close-up of the tracking visualization

To reference these images in the README.md file, use the following Markdown syntax:
![Description](screenshots/image_name.jpg)

You can replace the placeholders in the README.md file with your actual screenshots.
